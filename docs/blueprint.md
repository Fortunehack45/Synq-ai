# **App Name**: SynqAI

## Core Features:

- Secure Wallet Authentication: Integrate MetaMask and WalletConnect with SIWE for secure Ethereum and Base wallet authentication, without storing private keys.
- Responsive Dashboard: Display total balances, tokens, NFTs, portfolio charts, and transaction history in a responsive and intuitive dashboard.
- AI Chat Assistant: An AI-powered chat assistant using an LLM, providing grounded insights, real-time wallet analytics, risk scores, suggested actions, and on-chain citations as a tool.
- Risk Engine: Detect suspicious wallets, abnormal liquidity, or contract risks using Firebase Cloud Functions and ML heuristics.
- Transaction Simulation: Preview swaps, stakes, or transfers before confirmation through a simulation module.
- Admin Panel: Build an Admin Panel for logs, alerts, and AI decisions.
- Database integration: Utilize Firestore to store user data, wallets, holdings, tokens, NFTs, conversations, alerts, risk reports, and logs.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) for trust and intelligence.
- Background color: Light lavender (#EDE9F4) for a clean and calming interface.
- Accent color: Teal (#50A487) to highlight important information and actions.
- Body and headline font: 'Inter', a sans-serif font for a clean and modern look.
- Use clean, line-based icons to represent different crypto assets and functions.
- Implement a modular layout with clear sections for portfolio overview, transaction history, and AI insights.
- Use subtle transitions and animations to enhance user experience when loading data or interacting with the interface.
